# flipper-security-framework
Ethical RF, NFC, and IoT security assessment framework through the use of Flipper Zero and Python.

## Goals ##
- Provide a repeatable cybersecurity focused assessment of workflow, like a mini consulting playbook
- Collect and normalize Flipper captures, logs, and dumps
- Analyze results with lightweight Python scripts
- Generate clear findings + mitigation guidance

## Scope ##
### Included ###
- Sub-GHz: capture review, signal classification helpers, documentation workflow
- NFC/RFID: tag inventory + documentation workflow
- IR: capture organization + replay risk documentation workflow
- Basic IoT recon: device inventory + threat modeling templates
- Reporting: standardized findings + mitigation format

### Not Included ###
- Any exploitation instructions for unauthorized access
- Guidance or instructions for bypassing access control on real systems
